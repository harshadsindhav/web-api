Core Services

Overview

Gets the major, minor, and complete version numbers of the product.

Method

GET

Requires OAuth

No

URL

http://<hostname>/Saba/api/content/coreservice/version/

Return Values

{
    "@type": "com.saba.services.dto.VersionDTO",
    "version": "0605000042",
    "minor-version": "5",
    "major-version": "6"
}
