GET Methods

Overview

Returns the profile information based on certain criteria.

Requires OAuth

No

Method

GET

URL

http://<hostname>/v1/people?type=:type&f=(:searchFields)&q=(:query)

Calling Options

Table 3. Calling Options Calling Options
Name Description Default Value Data Type Required?
type User type [internal or external] internal string No
searchFields Indicate the field(s) that you intend to extract: [(state,country,username,status,suffix,title,city,zip,fname,lname, email,gender,homephone,workphone,fax,ethnicity,religion,is_manager, ss_no,job_title,terminated_on,person_type,mname,started_on, secret_answer,job_title_type,date_of_birth,person_no,special_user, secret_question,addr1,addr2,addr3,custom1,custom2,custom3, custom4,custom5,custom6,custom7,custom8,custom9,custom0,id] status,is_manager string No
query Search criteria [ex: status%3D%3DFull%20Time] status%3D%3DFull%20Time,is_manager%3D%3Dfalse string No
